# CleanQuote Profit Toolkit

Public, no-account storefront for the CleanQuote cleaning quote and profit workbook.

- Live calculator: browser-only estimate
- Product price: US$9 equivalent in Bitcoin
- Order transport: encrypted NIP-17 messages to the published seller key
- Delivery: encrypted, checksum-verified ZIP assembled in the buyer's browser after payment confirmation
- Paid product bytes are not committed to this public repository

The workbook is a decision-support tool, not accounting, tax, legal, employment, or financial advice. It does not guarantee profit or revenue.
